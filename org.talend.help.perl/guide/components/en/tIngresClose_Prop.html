tIngresClose

tIngresClose properties

Component family
Databases/Ingres
Function
tIngresClose closes the transaction committed in the connected DB.
Purpose
Close a transaction.
Basic settings
Component list
Select the tIngresConnection component in the list if more than one connection are planned for the current Job.
Advanced settings
tStatCatcher Statistics
Select this check box to collect log data at the component level.
Usage
This component is to be used along with Ingres components, especially with tIngresConnection and tIngresCommit.
