tAS400LastInsertId

tAS400LastInsertId properties

Component family
Databases
Function
tAS400LastInsertId fetches the last inserted ID from a selected AS400 Connection.
Purpose
tAS400LastInsertId obtains the primary key value of the record that was last inserted in an AS400 table by a user.
Basic settings
Schema and Edit Schema
A schema is a row description, i.e., it defines the number of fields to be processed and passed on to the next component. The schema is either built-in or stored remotely in the Repository.
 
 
Built-in: You create and store the schema locally for this component only. Related topic: How to set a built-in schema of Talend Integration Suite Studio User Guide.
 
 
Repository: You have already created the schema and stored it in the Repository. You can reuse it in various projects and job flow charts. Related topic: How to set a repository schema of Talend Integration Suite Studio User Guide.
