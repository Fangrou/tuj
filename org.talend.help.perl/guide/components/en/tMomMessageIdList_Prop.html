tMomMessageIdList

tMomMessageIdList Properties

Component family
Internet
Function
tMomMessageIdList fetches a message ID list from a queue on a Message-Oriented middleware system and passes it to the next component.
Purpose
tMomMessageIdList makes it possible to iterate on certain message IDs. It is usually used with tMomInput, for more information, see tMomInput Properties.
