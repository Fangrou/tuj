tInterbaseRollback

tInterbaseRollback properties

This component is closely related to tInterbaseCommit and tInterbaseConnection. It usually does not make much sense to use these components independently in a transaction.

Component family
Databases/Interbase
Function
tInterbaseRollback cancels the transaction committed in the connected DB.
Purpose
Avoids to commit part of a transaction involuntarily.
Basic settings
Component list
Select the tInterbaseConnection component in the list if more than one connection are planned for the current Job.
 
Close Connection
Clear this check box to continue to use the selected connection once the component has performed its task.
Advanced settings
tStatCatcher Statistics
Select this check box to collect log data at the component level.
Usage
This component is to be used along with Interbase components, especially with tInterbaseConnection and tInterbaseCommit.
Limitation
