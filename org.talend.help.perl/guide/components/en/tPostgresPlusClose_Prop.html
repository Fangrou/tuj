tPostgresPlusClose

tPostgresPlusClose properties

Component family
Databases/Postgres
Function
tPostgresPlusClose closes the transaction committed in the connected DB.
Purpose
Close a transaction.
Basic settings
Component list
Select the tPostgresPlusConnection component in the list if more than one connection are planned for the current Job.
Advanced settings
tStatCatcher Statistics
Select this check box to collect log data at the component level.
Usage
This component is to be used along with PostgresPlus components, especially with tPostgresPlusConnection and tPostgresPlusCommit.
