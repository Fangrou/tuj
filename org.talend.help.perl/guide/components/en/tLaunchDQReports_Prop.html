tLaunchDQReports

This component is available in the Palette of the studio on the condition that you have subscribed to the relevant edition of Talend Data Quality.

tLaunchDQReports properties

Component family
Data Quality
Function
tLaunchDQReports launches existing reports in Talend Data Quality studio without having to open the Talend Data Quality platform. It does the same action as the Generate report file menu item in the Talend Data Quality platform.
Purpose
tLaunchDQReports can communicate with Talend Data Quality in order to launch an analysis and save the results of the analysis in a database. Using this component, you can launch the generation of all types of reports: pdf, XML, or HTML.
For information about how to generate a report file, see Generating a report file in Talend Data Quality User Guide.
Basic settings
Talend Data Quality executable file
Path to the studio executable file.
 
Report filenames
Copy the technical file path of the report to generate from Talend Data Quality studio and paste it in this field. You can set several report filenames separated by commas.

You can display the technical file path of the report you want to generate in the Detail view area below DQRepository tree view in Talend Data Quality studio when you select the relevant report in the tree view.

Advanced settings
tStatCatcher Statistics
Select this check box to collect log data at the component level.
Usage
This component is used as a stand-alone component. It can launch reports from Talend Data Quality studio.
In order to be executable, a Job containing this component must have an access to Talend Data Quality studio executable file. If the Job is to be used in a jobserver, the latter needs to access Talend Data Quality studio executable file (e.g.TalendDataQuality-win32-x86.exe on Windows).
