tParAccelRollback

tParAccelRollback properties

This component is closely related to tParAccelCommit and tParAccelConnection. It usually doesn't make much sense to use these components independently in a transaction.

Component family
Databases
Function
Cancel the transaction commit in the connected DB.
Purpose
Avoids to commit part of a transaction involuntarily.
Basic settings
Component list
Select the tParAccelConnection component in the list if more than one connection are planned for the current job.
 
Close Connection
Clear this check box to continue to use the selected connection once the component has performed its task.
Advanced settings
tStatCatcher Statistics
Select this check box to collect log data at the component level.
Usage
This component is to be used along with ParAccel components, especially with tParAccelConnection and tParAccelCommit components.
Limitation
