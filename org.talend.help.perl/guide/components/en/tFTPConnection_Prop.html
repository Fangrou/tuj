tFTPConnection

tFTPConnection properties

Component family
Internet/FTP
Function
tFTPConnection opens an FTP connection in order that a transaction may be carried out.
Purpose
tFTPConnection allows you to open an FTP connection to transfer files in a single transaction.
Basic settings
Schema type
A schema is a row description, i.e., it defines the number of fields that will be processed and passed on to the next component. The schema is either Built-in or remote in the Repository.
 
 
Built-in: You create and store the schema locally for this component only. Related topic: How to set a built-in schema of Talend Integration Suite Studio User Guide.
 
 
Repository: You have already created the schema and stored it in the Repository. You can reuse it in various projects and job designs. Related topic: How to set a repository schema, in the Talend Integration Suite Studio User Guide.
 
Host
The FTP server IP address.
 
Port
The FTP server listening port number.
 
Username and Password
FTP user authentication data.
 
Connect mode
Select the mode: Active or Passive
Usage
This component is typically used as a single-component sub-job. It is used along with other FTP components.
