tMSSqlColumnList

Propriétés du tMSSqlColumnList

Famille de composant
Databases/MS SQL
Fonction
Le composant tMSSqlColumList effectue une opération d'itération dans toutes les colonnes d'une table donnée, grâce à une connexion MS SQL définie.
Objectif
Le tMSSqlColumList permet de faire une liste des libellés des colonnes d'une table MS SQL donnée.
Basic settings
Component list
Sélectionnez le composant tMSSqlConnection dans la liste si vous prévoyez d'ajouter plus d'une connexion à votre Job en cours.
 
Table name
Entrez le nom de la table.
Utilisation
Il faut utiliser ce composant en association avec les autres composants MS SQL, notamment avec le tMSSqlConnection.
Limitation
n/a
