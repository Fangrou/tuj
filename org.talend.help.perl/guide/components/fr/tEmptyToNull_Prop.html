tEmptyToNull

Propriétés du tEmptyToNull

Famille du composant
Processing
Fonction
Le composant tEmptyToNull transforme les champs vides d'un fichier ou d'une table en champs NULL dans une base de données.
Objectif
Le tEmptyToNull permet de remplacer des champs vides par des champs non définis produisant des valeurs NULL dans le composant de sortie.
Basic settings
 
Ce composant ne nécessite aucune configuration, il s'exécute automatiquement.
Advanced settings
tStatCatcher Statistics
Cochez cette case pour collecter les données de log au niveau du composant.
Utilisation
Ce composant est généralement utilisé comme composant intermédiaire dans un flux de données, il nécessite donc un composant en entrée et un composant en sortie. Le composant en sortie doit obligatoirement être de type base de données.
