tLastRegexlibExpressions

Ce composant est disponible dans la Palette du studio uniquement si vous avez souscrit à l'édition correspondante de Talend Data Quality.

Propriétés du tLastRegexlibExpressions

Famille de composant
Data Quality
Fonction
Le tLastRegexlibExpressions se connecte à un service Web, sur http://regexlib.com afin d'obtenir les expressions les N plus récentes parmi la liste d'expressions régulières.
Objectif
Le tLastRegexlibExpressions retourne un ensemble de données contenant des informations à propos des expressions régulières N les plus récemment ajoutées à la bibliothèque qui correspondent à la requête envoyée à http://regexlib.com.
Basic settings
Schema et Edit Schema
Ces champs sont en lecture seule. Le schéma du composant contient les champs suivants : Title, Expression, Description, Matches, Non-Matches, Author, Rating (Titre, Expression, Description, Correspondances, Non-correspondaces, Auteur, Note).
 
Max Rows
Saisissez le nombre maximal d'expressions que vous souhaitez obtenir.
Advanced settings
tStatCatcher Statistics
Cochez cette case pour collecter les métadonnées de process du Job, aussi bien au niveau du Job qu'au niveau de chaque composant.
Utilisation
Ce composant est un composant d'entrée. Il requiert un flux de sortie, habituellement un fichier .csv. Vous pourrez importer ensuite toutes les expressions collectées depuis un fichier .csv avec un format correct dans le studio de Talend Data Quality.
Pour plus d'informations sur l'import des modèles, consultez le Guide utilisateur Talend Data Quality (en anglais).
