tDenormalize

Propriétés du tDenormalize

Famille de composant
Processing/Fields
Fonction
Le composant tDenormalize dénormalise un flux entrant en fonction d'une colonne.
Objectif
Le tDenormalize permet de synthétiser un flux entrant.
Basic settings
Schema type et Edit Schema
Un schéma est une description de lignes, i.e., il définit le nombre de champs qui sont traités et passés au composant suivant. Le schéma est soit local (built-in) soit distant dans le Repository. En Perl, le schéma est en lecture seule et obligatoirement en Built-In.
 
 
Built-in : Le schéma est créé et conservé pour ce composant uniquement. Voir également : Paramétrer un schéma built-in du Guide utilisateur de Talend Open Studio.

Java uniquement
To denormalize
Ce tableau vous permet de choisir les paramètres de dénormalisation des colonnes.
Column : Sélectionnez la colonne à dénormaliser.
Delimiter : Saisissez, entre guillemets doubles, le séparateur que vous souhaitez utiliser.
Merge same value : Cochez cette case si vous voulez fusionner les valeurs identiques.

Perl uniquement
Column to denormalize
Sélectionnez la colonne du flux entrant sur laquelle est basée la dénormalisation (inclus dans la clé).

Java uniquement
Group by
Sélectionnez une ou plusieurs colonnes à regrouper. Nous vous conseillons de supprimer du schéma les colonnes qui ne sont pas utilisées avant de procéder au traitement.

Perl uniquement
Item Separator
Renseignez le séparateur délimitant les données du flux dénormalisé.
Advanced settings
 
 

Perl uniquement
Deduplicate items
Supprime les doublons lorsque les valeurs dénormalisées sont concaténées.
 
tStatCatcher Statistics
Cochez cette case pour collecter les données de log au niveau du composant.
Utilisation
Ce composant peut être utilisé en étape intermédiaire dans un flux de données.
Limitation
Notez que ce composant peut changer l'ordre du flux entrant en Java et en Perl.
