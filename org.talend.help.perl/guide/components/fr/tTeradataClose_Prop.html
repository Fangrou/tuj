tTeradataClose

Propriétés du tTeradataClose

Famille de composant
Databases/Teradata
Fonction
Le composant tTeradataClose ferme la connexion à une base de données connectée.
Objectif
Ce composant permet de fermer une connexion à la base de données Teradata.
Basic settings
Component list
S'il y a plus d'une connexion dans le Job en cours, sélectionnez le composant tTeradataConnection dans la liste.
Advanced settings
tStatCatcher Statistics
Cochez cette case pour collecter les données de log au niveau du composant.
Utilisation
Ce composant est généralement utilisé comme composant de début. Il nécessite un composant de sortie.
Limitation
n/a
