tSAPRollback

Propriétés du tSAPRollback

Ce composant est étroitement lié aux composants tSAPCommit et tSAPConnection. Ils sont généralement utilisés ensemble lors de transactions..

Component family
Business/SAP
Function
tSAPRollback annule la transaction dans le SAP connecté.
Purpose
tSAPRollback évite le commit de transaction involontaire.
Basic settings
Component list
Sélectionnez le composant de connexion tSAPConnection dans la liste si vous prévoyez d'ajouter plus d'une connexion à votre Job en cours.
 
Release Connection
Décochez cette case afin de continuer à utiliser la connexion sélectionnée, une fois que le composan a exécuté sa tâche.
Advanced settings
tStatCatcher Statistics
Cochez cette case pour collecter les données de log au niveau du composant.
Usage
Ce composant est généralement utilisé avec des composants SAP et notamment avec les composants tSAPConnection et tSAPCommit.
Limitation
n/a
